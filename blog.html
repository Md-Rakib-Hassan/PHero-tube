1. Discuss the scope of 'var', 'let', and 'const'.

'var' is globally scoped or function scoped its mean if I declared a variable as 'var' in a function it will accessible from anywhere of that function and if I declared it any where without function then it hoisted the declaration at the top as undefined and accessible to anywhere. And it can be updated and redeclared as well.


'let' is block scoped its mean if I declared a variable as 'let' then it only can accessible within that block. And it can be updated but it cannot be redeclared within same scoped.


'const' also a block scoped as like let but its value can not be changed after the initialization. Its maintain a constant value that given with the declaration.

2. Tell us the use cases of null and undefined.

If we declared a variable but not initialized the value or not declared the variable and want to access it then it gives undefined so undefined is used to represent an uninitialized or missing variable, where null means an empty value or a blank value. Usually, null use to represent an object or array that has no value, and the type of null is object so null is used to represent an object with no parameter or values and we also can give a variable value as null to understand that the value of that variable is empty like let x=null;
